# React Native Redux Example

To get started:

1. Clone repo   
```
git clone https://github.com/react-native-training/basic-react-native-redux-example.git
```

2. Install dependencies   
```
npm i || yarn
```

3. Start packager   
```
npm start
```

4. Run on device of choice.
